# Sportshop

Интернет-магазин спортивных товаров.

## Структура проекта

- `services/` — микросервисы 
- `frontend/` — React-приложение
- `shared-types/` — общие TypeScript-типы
- `docker-compose.yml` — оркестрация контейнеров
